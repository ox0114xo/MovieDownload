🛠️ 2. 基礎核心與魔法師安裝

首次使用請先輸入第一行指令賦予手機儲存權限。接著安裝最強下載核心 yt-dlp 與影音融合工具 ffmpeg。

termux-setup-storage
pkg update && pkg upgrade -y
pkg install python ffmpeg -y
pip install -U yt-dlp

⚙️ 3. 全自動存檔與防呆設定 (強烈建議)

我們幫你設計了最貼心的「管家機制」，貼上這段指令後，您將享有以下特權：

  • 自動防錯：如果影片標題太長，會自動縮減至 50個字以內，防止手機存檔失敗報錯。
  • 相容性 100%：所有影片都會強制轉為 MP4 格式，確保任何手機相簿都能直接播放。
  • 免找檔案：下載完畢會自動存入手機的 Download (下載) 資料夾。
mkdir -p ~/.config/yt-dlp
echo '-o /sdcard/Download/%(title)s.%(ext)s' > ~/.config/yt-dlp/config
echo '--merge-output-format mp4' >> ~/.config/yt-dlp/config
echo '--trim-filenames 50' >> ~/.config/yt-dlp/config
echo '--remux-video mp4' >> ~/.config/yt-dlp/config

🚀 4. 懶人必殺技：智能分享攔截腳本

遇到 B站 或 小紅書 這種「標題+網址」的混合分享也不怕！設定好這個腳本，以後免打字一鍵下載。

💡 使用秘訣：在 App 點擊「分享給 Termux」，若跳出黑色詢問視窗，請點擊右下角 EDIT 即可！

mkdir -p ~/bin

cat << 'EOF' > ~/bin/termux-file-editor
clear
echo "🚀 收到分享內容，正在自動過濾網址..."
CONTENT=$(cat "$1")
URL=$(echo "$CONTENT" | grep -oE 'https?://[a-zA-Z0-9./?=_%:-]+' | head -n 1)

if [ -n "$URL" ]; then
    echo "🔗 成功提取網址: $URL"
    yt-dlp "$URL"
else
    echo "❌ 找不到有效網址！"
fi
echo ""
read -p "✅ 任務結束！請按 Enter 鍵關閉視窗..."
EOF

chmod +x ~/bin/termux-file-editor

🪄 5. 進階魔法指令 (高階玩家專屬)

除了無腦分享下載，你也可以手動輸入「密語」來達成更高級的任務（請將指令中的 "網址" 替換為真實連結）：

  • 🎧 只要聲音 (MP3 模式)：
    把翻唱影片或 Podcast 直接抽成 MP3 音樂。 
    yt-dlp -x --audio-format mp3 "網址"
  • 📦 影分身之術 (整包播放清單下載)：
    遇到 YouTube 整套教學或音樂精選集，一鍵全部排隊下載。 
    yt-dlp --yes-playlist "播放清單網址"
  • 📉 手機容量救星 (限制最高畫質 1080p)：
    不想被 4K 影片塞爆手機容量？強制它只抓 1080p 版本。 
    yt-dlp -S "res:1080" "網址"
