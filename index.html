🛠️ 2. 基礎核心與魔法師安裝

首次使用請先輸入第一行指令賦予手機儲存權限。接著安裝最強下載核心 yt-dlp 與影音融合工具 ffmpeg。

termux-setup-storage
pkg update && pkg upgrade -y
pkg install python ffmpeg -y
pip install -U yt-dlp

⚙️ 3. 一鍵全自動防呆設定 (初次必用)

我們幫你設計了最貼心的「管家機制」，貼上這段指令後，您將享有以下特權：

  • 自動防錯：如果影片標題太長，會自動縮減至 50個字以內，防止手機存檔失敗報錯。
  • 相容性 100%：所有影片都會強制轉為 MP4 格式，確保任何手機相簿都能直接播放。
  • 免找檔案：下載完畢會自動存入手機的 Download (下載) 資料夾。
mkdir -p ~/.config/yt-dlp
echo '-o /sdcard/Download/%(title)s.%(ext)s' > ~/.config/yt-dlp/config
echo '--merge-output-format mp4' >> ~/.config/yt-dlp/config
echo '--trim-filenames 50' >> ~/.config/yt-dlp/config
echo '--remux-video mp4' >> ~/.config/yt-dlp/config

🚀 4. 全能必殺技：智能分享攔截腳本 (Python 防彈版)

為了解決安卓系統背景分享時容易發生「權限報錯」的問題，我們捨棄傳統做法，直接寫入純 Python 引擎。複製貼上這段「終極代碼」，完全免疫權限報錯，一鍵通殺純網址與混合圖文分享！

💡 使用秘訣：在各大 App 看到影片，點擊「分享給 Termux」，若跳出黑色詢問視窗，請點擊右下角 EDIT 即可自動下載！

# 1. 強制清除所有卡死的舊檔案
rm -f ~/bin/termux-url-opener
rm -f ~/bin/termux-file-editor
mkdir -p ~/bin

# 2. 寫入無敵的 Python 攔截引擎
cat << 'EOF' > ~/bin/termux-url-opener
#!/data/data/com.termux/files/usr/bin/python
import sys, os, re

print("\n🚀 正在啟動純淨下載引擎...")
try:
    content = sys.argv[1]
    
    # 自動判斷是純網址還是圖文混合
    if os.path.isfile(content):
        with open(content, 'r', encoding='utf-8') as f:
            content = f.read()
            
    # 精準捕捉網址
    urls = re.findall(r'https?://[a-zA-Z0-9./?=_%:-]+', content)
    if urls:
        url = urls[0]
        print(f"🔗 鎖定目標網址: {url}\n")
        # 底層直連，免疫權限與路徑遺失問題
        os.system(f'"{sys.executable}" -m yt_dlp "{url}"')
    else:
        print("❌ 找不到有效網址！")
except Exception as e:
    print(f"❌ 發生致命錯誤: {e}")

input("\n✅ 任務結束！請按 Enter 鍵關閉視窗...")
EOF

# 3. 賦予權限並共用引擎
chmod +x ~/bin/termux-url-opener
cp ~/bin/termux-url-opener ~/bin/termux-file-editor

🪄 5. 進階魔法指令 (高階玩家專屬)

除了無腦分享下載，你也可以在 Termux 裡手動輸入「密語」來達成更高級的任務（請將指令中的 "網址" 替換為真實連結）：

  • 🎧 只要聲音 (MP3 模式)：
    把翻唱影片或 Podcast 直接抽成 MP3 音樂。 
    yt-dlp -x --audio-format mp3 "網址"
  • 📦 影分身之術 (整包播放清單下載)：
    遇到 YouTube 整套教學或音樂精選集，一鍵全部排隊下載。 
    yt-dlp --yes-playlist "播放清單網址"
  • 📉 手機容量救星 (限制最高畫質 1080p)：
    不想被 4K 影片塞爆手機容量？強制它只抓 1080p 版本。 
    yt-dlp -S "res:1080" "網址"

🆘 疑難排解 (Bug 急救站)

  • 突然跳紅字不能下載了？
    網站防抓取機制更新了。打開 Termux 輸入 pip install -U yt-dlp 升級到最新版，通常就能滿血復活。
  • 手動下載指令：不想用分享按鈕，也可以在 Termux 直接輸入 yt-dlp "你要下載的網址"。
