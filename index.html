🛠️ 2. 基礎核心與魔法師安裝

首次使用請先輸入第一行指令賦予手機儲存權限。接著安裝最強下載核心 yt-dlp 與影音融合工具 ffmpeg。

termux-setup-storage
pkg update && pkg upgrade -y
pkg install python ffmpeg -y
pip install -U yt-dlp

⚙️ 3. 全自動存檔與轉檔設定

貼上這段指令，以後下載的影片都會自動轉為 MP4、縮短檔名，並直接送進手機的 Download 資料夾。

mkdir -p ~/.config/yt-dlp
echo '-o /sdcard/Download/%(title)s.%(ext)s' > ~/.config/yt-dlp/config
echo '--merge-output-format mp4' >> ~/.config/yt-dlp/config
echo '--trim-filenames 50' >> ~/.config/yt-dlp/config
echo '--remux-video mp4' >> ~/.config/yt-dlp/config

🚀 4. 終極分享攔截腳本 (B站/小紅書必備)

這個腳本能自動過濾出帶有中文字分享裡的乾淨網址。

💡 秘訣：在 App 點擊「分享給 Termux」，若跳出黑色詢問視窗，請點擊右下角 EDIT！

mkdir -p ~/bin

cat << 'EOF' > ~/bin/termux-file-editor
clear
echo "🚀 收到分享內容，正在自動過濾網址..."
CONTENT=$(cat "$1")
URL=$(echo "$CONTENT" | grep -oE 'https?://[a-zA-Z0-9./?=_%:-]+' | head -n 1)

if [ -n "$URL" ]; then
    echo "🔗 成功提取網址: $URL"
    yt-dlp "$URL"
else
    echo "❌ 找不到有效網址！"
fi
echo ""
read -p "✅ 任務結束！請按 Enter 鍵關閉視窗..."
EOF

chmod +x ~/bin/termux-file-editor
✅ 程式碼已複製！
