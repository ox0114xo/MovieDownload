🛠️ 2. 打底子：安裝核心與魔法師

打開剛裝好的 Termux，首次使用請先輸入第一行指令賦予手機儲存權限（會跳出詢問視窗，請按允許）。接著安裝最強下載核心 yt-dlp 與影音融合工具 ffmpeg。

💡 提示： 請點擊「複製指令」，貼入 Termux 後按 Enter 執行，跑到出現 $ 符號代表完成。

termux-setup-storage
pkg update && pkg upgrade -y
pkg install python ffmpeg -y
pip install -U yt-dlp

⚙️ 3. 裝管家：全自動存檔防呆設定

我們幫你設計了最貼心的「管家機制」，貼上這段指令後，您將享有以下特權：

  • 自動防錯：如果影片標題太長，會自動縮減至 50個字以內，防止手機存檔失敗報錯。
  • 相容性 100%：所有影片都會強制轉為 MP4 格式，確保任何手機相簿都能直接播放。
  • 免找檔案：下載完畢會自動存入手機的 Download (下載) 資料夾。
mkdir -p ~/.config/yt-dlp
echo '-o /sdcard/Download/%(title)s.%(ext)s' > ~/.config/yt-dlp/config
echo '--merge-output-format mp4' >> ~/.config/yt-dlp/config
echo '--trim-filenames 50' >> ~/.config/yt-dlp/config
echo '--remux-video mp4' >> ~/.config/yt-dlp/config

🚀 4. 裝引擎：智能分享攔截腳本 (終極防彈版)

這段是最核心的「分享接管引擎」！它可以解決安卓系統嚴苛的權限限制。
複製貼上這段代碼，就能一鍵通殺**「純網址 (如 IG/YT)」**與**「圖文混合 (如 B站/小紅書)」**！

💡 終極秘訣：以後在各大 App 看到影片，點擊「分享給 Termux」。若跳出黑色詢問視窗，請點擊右下角 EDIT，它就會自動幫你飆速下載！

# 1. 強制清除舊的設定，砍掉重練
rm -f ~/bin/termux-url-opener
rm -f ~/bin/termux-file-editor
mkdir -p ~/bin

# 2. 寫入全新的「替身喚醒引擎」(完全免疫路徑遺失與權限報錯)
cat << 'EOF' > ~/bin/termux-url-opener
#!/data/data/com.termux/files/usr/bin/bash
exec bash -l -c '
clear
echo "🚀 啟動終極下載引擎..."

CONTENT="$1"
if [ -f "$CONTENT" ]; then
    CONTENT=$(cat "$CONTENT")
fi

URL=$(echo "$CONTENT" | grep -oE "https?://[a-zA-Z0-9./?=_%:-]+" | head -n 1)

if [ -n "$URL" ]; then
    echo "🔗 鎖定網址: $URL"
    yt-dlp "$URL"
else
    echo "❌ 找不到有效網址！"
fi

echo ""
read -p "✅ 任務結束！請按 Enter 鍵關閉視窗..." dummy
' _ "$1"
EOF

# 3. 賦予執行權限，並讓純網址和圖文共用這個引擎
chmod +x ~/bin/termux-url-opener
cp ~/bin/termux-url-opener ~/bin/termux-file-editor

🪄 5. 進階魔法指令 (隱藏大招)

除了無腦分享下載，你也可以在 Termux 裡手動輸入「密語」來達成更高級的任務（請將指令中的 "你要下載的網址" 替換為真實連結，並保留雙引號）：

  • 🎧 只要聲音 (MP3 模式)：
    把翻唱影片或 Podcast 直接抽成 MP3 音樂。 
    yt-dlp -x --audio-format mp3 "你要下載的網址"
  • 📦 影分身之術 (整包播放清單下載)：
    遇到 YouTube 整套教學或音樂精選集，一鍵全部排隊下載。 
    yt-dlp --yes-playlist "你要下載的網址"
  • 📉 手機容量救星 (限制最高畫質 1080p)：
    不想被 4K 影片塞爆手機容量？強制它只抓 1080p 版本。 
    yt-dlp -S "res:1080" "你要下載的網址"

🆘 疑難排解 (Bug 急救站)

  • 突然跳紅字不能下載了？
    網站防抓取機制更新了。打開 Termux 輸入 pip install -U yt-dlp 升級到最新版，通常就能滿血復活。
  • 不想用分享按鈕下載？
    隨時可以在 Termux 直接輸入 yt-dlp "你要下載的網址"。
