📱 影音下載神器

一鍵抓取 IG/FB/YT/抖音，最高畫質、無廣告

第 1 步：下載正確的 App

⚠️ 不要用 Google Play 商店版！請點擊下方連結下載官方最新版：

👉 F-Droid 官方下載 Termux APK

第 2 步：安裝核心引擎 (新手必做)

打開 Termux，直接複製下方整段貼入，過程中若跳出 [Y/n] 請直接按 Enter：

termux-setup-storage
pkg update && pkg upgrade -y
pkg install python ffmpeg -y
pip install -U yt-dlp

*這會幫你安裝 Python 下載器與影片合成工具。

第 3 步：全自動存檔設定

這段指令會讓影片下載後，自動存入你手機的 Download (下載) 資料夾：

mkdir -p ~/.config/yt-dlp
echo '-o /sdcard/Download/%(title)s.%(ext)s' > ~/.config/yt-dlp/config
echo '--merge-output-format mp4' >> ~/.config/yt-dlp/config
echo '--trim-filenames 50' >> ~/.config/yt-dlp/config
echo '--remux-video mp4' >> ~/.config/yt-dlp/config

第 4 步：植入「分享下載」大招

貼上這段後，以後只要在 YouTube/IG 按下「分享給 Termux」，點 EDIT 就會自動下載：

rm -f ~/bin/termux-url-opener
rm -f ~/bin/termux-file-editor
mkdir -p ~/bin
cat << 'EOF' > ~/bin/termux-url-opener
#!/data/data/com.termux/files/usr/bin/bash
exec bash -l -c '
clear
echo "🚀 啟動終極下載引擎..."
if ! command -v yt-dlp &> /dev/null; then
    echo "❌ 偵測到尚未安裝核心引擎！"
    echo "請回到教學網頁完成「第 2 步」安裝。"
    read -p "按 Enter 鍵結束..." dummy
    exit
fi
CONTENT="$1"
[ -f "$CONTENT" ] && CONTENT=$(cat "$CONTENT")
URL=$(echo "$CONTENT" | grep -oE "https?://[a-zA-Z0-9./?=_%:-]+" | head -n 1)
if [ -n "$URL" ]; then
    echo "🔗 鎖定網址: $URL"
    yt-dlp "$URL"
else
    echo "❌ 找不到有效網址！"
fi
echo ""
read -p "✅ 任務結束！請按 Enter 鍵關閉..." dummy
' _ "$1"
EOF
chmod +x ~/bin/termux-url-opener
cp ~/bin/termux-url-opener ~/bin/termux-file-editor

💡 小秘訣

  • 出現白色詢問框？ 請去手機設定將 Termux 的「顯示在其他應用程式上層」權限打開。
  • 想下載 MP3？ 在 Termux 輸入：
    yt-dlp -x --audio-format mp3 "網址"
✅ 已複製到剪貼簿
