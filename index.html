🛠️ 2. 安裝核心引擎 (打底子)

打開 Termux，直接複製下方整段指令貼入後按 Enter。過程中若跳出詢問請一路按 Enter 或輸入 y。

termux-setup-storage
pkg update && pkg upgrade -y
pkg install python ffmpeg -y
pip install -U yt-dlp

* 安裝 FFmpeg 是解決「有聲無影」或「有影無聲」問題的關鍵。

⚙️ 3. 全自動存檔防呆設定

這段指令會設定影片自動存入手機的 Download 資料夾，並自動縮減過長的標題字數。

mkdir -p ~/.config/yt-dlp
echo '-o /sdcard/Download/%(title)s.%(ext)s' > ~/.config/yt-dlp/config
echo '--merge-output-format mp4' >> ~/.config/yt-dlp/config
echo '--trim-filenames 50' >> ~/.config/yt-dlp/config
echo '--remux-video mp4' >> ~/.config/yt-dlp/config

🚀 4. 植入「分享下載」大招 (防彈版)

這是最關鍵的一步！它會解決背景分享時的「權限不足」與「找不到路徑」問題。

rm -f ~/bin/termux-url-opener
rm -f ~/bin/termux-file-editor
mkdir -p ~/bin
cat << 'EOF' > ~/bin/termux-url-opener
#!/data/data/com.termux/files/usr/bin/bash
exec bash -l -c '
clear
echo "🚀 影音合體引擎啟動..."
CONTENT="$1"
[ -f "$CONTENT" ] && CONTENT=$(cat "$CONTENT")
URL=$(echo "$CONTENT" | grep -oE "https?://[a-zA-Z0-9./?=_%:-]+" | head -n 1)
if [ -n "$URL" ]; then
    echo "🔗 鎖定目標: $URL"
    yt-dlp -f "bv+ba/b" --merge-output-format mp4 --remux-video mp4 "$URL"
    echo "✅ 下載完畢！請至相簿 Download 查看。"
else
    echo "❌ 找不到有效網址！"
fi
read -p "任務結束，請按 Enter 關閉..." dummy
' _ "$1"
EOF
chmod +x ~/bin/termux-url-opener
cp ~/bin/termux-url-opener ~/bin/termux-file-editor

⚠️ 重要權限：避免「白色詢問框」

若在下載時跳出系統權限提示，請執行：

  • 進入手機 **「設定」** ➡️ **「應用程式管理」**。
  • 找到 **「Termux」** ➡️ **「進階」**。
  • 開啟 **「顯示在其他應用程式上層」** (懸浮視窗權限)。

🪄 5. 進階魔法指令

  • 只要 MP3 音樂：
    yt-dlp -x --audio-format mp3 "網址"
  • 下載整包播放清單：
    yt-dlp --yes-playlist "網址"
  • 限制 1080p 畫質 (省空間)：
    yt-dlp -S "res:1080" "網址"
