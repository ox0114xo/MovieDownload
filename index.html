📱 Termux 終極影音下載神器

支援 IG, Threads, FB, Bilibili, YouTube 一鍵無廣告下載

🛠️ 1. 基礎核心安裝

打開 Termux，首先賦予手機存取權限，並安裝最強下載器 yt-dlp 與影音融合魔法師 ffmpeg。

termux-setup-storage
pkg update && pkg upgrade -y
pkg install python ffmpeg -y
pip install -U yt-dlp

⚙️ 2. 全自動存檔設定 (免搬運)

設定這段指令後，下載的影片會自動轉成 MP4，縮短檔名，並直接存入手機相簿的 Download 資料夾。

mkdir -p ~/.config/yt-dlp
echo '-o /sdcard/Download/%(title)s.%(ext)s' > ~/.config/yt-dlp/config
echo '--merge-output-format mp4' >> ~/.config/yt-dlp/config
echo '--trim-filenames 50' >> ~/.config/yt-dlp/config
echo '--remux-video mp4' >> ~/.config/yt-dlp/config

🚀 3. 終極必殺技：一鍵分享下載腳本

這是一段攔截腳本，能自動過濾 B站/小紅書 帶有中文字的分享連結，精準抓出網址並下載。

💡 用法：在 App 點擊分享給 Termux，遇到彈出視窗請點擊右下角的 EDIT 即可開始下載！

mkdir -p ~/bin

cat << 'EOF' > ~/bin/termux-file-editor
clear
echo "🚀 收到分享內容，正在自動過濾網址..."
CONTENT=$(cat "$1")
URL=$(echo "$CONTENT" | grep -oE 'https?://[a-zA-Z0-9./?=_%:-]+' | head -n 1)

if [ -n "$URL" ]; then
    echo "🔗 成功提取網址: $URL"
    yt-dlp "$URL"
else
    echo "❌ 找不到有效網址！"
fi
echo ""
read -p "✅ 任務結束！請按 Enter 鍵關閉..."
EOF

chmod +x ~/bin/termux-file-editor

🆘 遇到問題怎麼辦？

  • 無法下載或報錯？ 網站通常改版了，輸入 pip install -U yt-dlp 更新即可滿血復活。
  • 純網址分享： 如果你只複製了純網址，也可以直接在 Termux 輸入 yt-dlp "你的網址" 進行手動下載。
